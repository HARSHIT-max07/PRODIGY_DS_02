# 🚢 PRODIGY_DS_02 — Data Cleaning and Exploratory Data Analysis (Titanic Dataset)

## 1. Overview

This project focuses on performing data cleaning and exploratory data analysis (EDA) on the Titanic dataset. The objective is to explore relationships between variables and identify patterns that influenced passenger survival. Data visualization techniques were used to better understand trends and insights from the dataset.

---

## 2. Objective

The main objectives of this project are:

1. Perform data cleaning and preprocessing
2. Handle missing values
3. Explore relationships between variables
4. Identify survival patterns and trends
5. Visualize data using charts and graphs

---

## 3. Dataset

Dataset Used: Titanic Dataset (Kaggle)

The dataset contains passenger information such as:

1. PassengerId
2. Pclass (Passenger Class)
3. Name
4. Sex
5. Age
6. SibSp (Siblings/Spouses aboard)
7. Parch (Parents/Children aboard)
8. Fare
9. Embarked
10. Survived

## 7. Exploratory Data Analysis

The following analyses were performed:

1. Survival rate by gender
2. Survival rate by passenger class
3. Age distribution of passengers
4. Fare distribution
5. Survival based on embarkation port
6. Correlation between variables

---

## 8. Visualizations

The following visualizations were created:

1. Bar charts
2. Count plots
3. Histograms
4. Heatmap correlation matrix
5. Distribution plots

---

## 9. Key Insights

1. Females had higher survival rate than males
2. First-class passengers had higher survival rate
3. Younger passengers had better survival chances
4. Higher fare passengers showed better survival probability
5. Passenger class played an important role in survival
